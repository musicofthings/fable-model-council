#The council

orchestrator

Fable 5

or Opus 4.8

Talks to you, plans, routes work to the right tier, verifies results before reporting them, and synthesizes everything back into one answer.

worker

Opus 4.8

or Sonnet 5

An autonomous agent with bash and a text editor, path-confined to its own workspace. Writes code, runs commands, and reports back with evidence.

routine

Sonnet 5

or Haiku 4.5

Fast, cheap, tool-free text tasks: drafts, summaries, rewrites, reformatting, classification, boilerplate.

#Features

Autonomous goal loops

/goal sets a standing objective; /loop [n] [budget] pursues it hands-free — the top tier reasons and delegates each iteration, the cheaper tiers do the work, and an independent fresh-context verifier must confirm any "goal achieved" claim before the loop stops.

Interactive model picker

Choose a model per role at startup. Prompts, tool descriptions, pricing, and fallback wiring all adapt to the selection.

Parallel subagents

Tool calls issued in one orchestrator turn run concurrently, with tagged, line-buffered progress ([w1], [s2.t1]) instead of interleaved streams.

Dynamic workflows

The run_workflow tool executes multi-stage pipelines composed at runtime — tasks within a stage run in parallel, and later stages automatically receive all earlier outputs.

Prompt caching

A rotating window of cache breakpoints follows the growing conversation, and the worker's inner loop caches its own history. /usage itemizes cache reads and writes per model.

Refusal fallbacks

A Fable 5 orchestrator ships with server-side fallbacks to Opus 4.8, so a safety-classifier false positive is transparently re-served instead of failing the turn.

Sandboxed worker

Point the council at any folder on your machine at startup (default ~/council-workspace) — file edits are path-confined to it, and shell commands run under Git Bash with a 180 s timeout.

pip install anthropic

# auth: ANTHROPIC_API_KEY env var, or an `ant auth login` profile
git clone https://github.com/musicofthings/fable-model-council.git
cd fable-model-council
python council.py
